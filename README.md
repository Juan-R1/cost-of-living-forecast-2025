# Texas Cost of Living Forecast 2025

A data analytics portfolio project focused on forecasting rental and cost trends across major Texas cities.

## Status

Portfolio cleanup in progress. The repository is organized with data, notebook, and dashboard folders. The next step is to add final screenshots, data-source notes, and a results section.

## Project Goal

Compare city-level cost trends and create a clear forecasting workflow that can support budgeting, relocation, and business planning conversations.

## Tools

- Python
- pandas
- Jupyter Notebook
- Time series forecasting
- Tableau or dashboard visualization
- Git/GitHub

## Repository Structure

- `Data/` - source or cleaned data files
- `Notebooks/` - analysis and forecasting workflow
- `Dashboard/` - dashboard files or exports
- `README.md` - portfolio summary

## Workflow

1. Collect city-level cost or rental data.
2. Clean and prepare the dataset.
3. Build a forecasting workflow.
4. Compare city-level trends.
5. Present findings in a dashboard or visual summary.
